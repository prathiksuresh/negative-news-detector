# Negative News Detector

## Overview
Negative News Detector is a Python automation tool designed to identify potentially negative vendor-related news articles.

The script reads news article links, extracts article content, and checks for predefined negative keywords to classify the article as:

- Negative
- Generic

This helps risk monitoring teams prioritize high-risk vendor news.

## Features
- Automated article review
- Keyword-based classification
- Excel input and output
- Detects negative risk signals in vendor news

## Technologies
- Python
- BeautifulSoup
- Pandas
